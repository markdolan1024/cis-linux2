Ionic Notes

Ionic is a cool platform for developing hybrid mobile apps, or apps that are completely written as a web page. Ionic has many built in features that are built on top of AngularJS, so using Angular with Ionic is usually best. I wrote Sciencetap with this and found it was a great way to learn angular as well.
